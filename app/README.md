# FlutFire

A complete Flutter project with Firebase Authentication, Firestore, Push Notifications and API calls.

# Technologies

- App: Flutter 2.2.3
- Server: NodeJs v16.5.0

# App
## Setup

```bash
make install
```

## Running

```bash
make run
```

## Build

```bash
make build-android
make build-ios
```

# Server
## Setup

```bash
npm install
```

## Running

```bash
npm start
```

## Deploy

```bash
npm run deploy
```
